# gmat
Playing around with NASA General Mission Analysis Tool

## simulate.py
```
usage: simulate.py [-h] [--crew_count CREW_COUNT] [--script SCRIPT] [--crew_avg_mass CREW_AVG_MASS] [--spacecraft_dry_mass SPACECRAFT_DRY_MASS]

options:
  -h, --help            show this help message and exit
  --crew_count CREW_COUNT, -c CREW_COUNT
                        The number of crew on the spacecraft
  --script SCRIPT, -s SCRIPT
                        The path to the GMAT script
  --crew_avg_mass CREW_AVG_MASS
                        units: kg
  --spacecraft_dry_mass SPACECRAFT_DRY_MASS
                        units: kg
```
